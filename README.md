## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/en/)
- [Pnpm](https://pnpm.io/)

### Installation

1. Clone the repo
   ```sh
   git clone
   ```
2. Install NPM packages

```sh
  pnpm install
```

4. Start the project

```sh
  pnpm dev --port 3000
```

6. Open your browser and go to `http://localhost:3000/`
